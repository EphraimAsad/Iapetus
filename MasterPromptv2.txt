- measure change in:
  - final predicted log CFU/g
  - threshold exceedance probability
  - recommended max shelf life

Output structure should include:
- baseline values

PART E — RATKOWSKY TEMPERATURE RESPONSE
==================================================

Add explicit temperature-response modelling for mu_max.

Create:
- backend/app/utils/temperature_models.py

Implement Ratkowsky-style relationship:

sqrt(mu_max) = b * (T - Tmin)

Therefore:
mu_max = (b * (T - Tmin))^2

Requirements:
- clamp mu_max to zero if T <= Tmin
- allow configurable Tmin and b
- integrate this into estimate_mu_max(...)
- make low/chilled temperatures suppress growth strongly
- make warmer abuse temperatures increase growth rate

Keep implementation simple, explicit, and modular.

Do not overcomplicate with multiple organism libraries at this stage.
Focus on Listeria monocytogenes use case for MVP.
